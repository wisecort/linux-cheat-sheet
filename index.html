🔐Permissões
chmod, chown, bits especiais (SUID/SGID/sticky) e ACLs.
chmod — Permissões Numéricas
Formato: rwxrwxrwx → dono | grupo | outros  |  r=4  w=2  x=1
chmod 755 filerwxr-xr-x (típico para executáveis)
chmod 644 filerw-r--r-- (típico para arquivos)
chmod 600 filerw------- (chave SSH privada)
chmod 700 dirrwx------ (diretório privado)
chmod +x script.shAdiciona permissão de execução
chmod -w fileRemove permissão de escrita
chmod -R 755 dir/Aplica recursivamente
chmod 4755 fileSUID — executa como dono do arquivo
chmod 2755 dirSGID — herda grupo do diretório pai
chmod 1777 dirSticky bit — só dono pode apagar (ex: /tmp)
chmod na prática
$ ls -l script.sh
-rw-r--r-- 1 matheus devs 512 Mar 26 script.sh
# sem permissão de execução — vamos adicionar
$ chmod +x script.sh
$ ls -l script.sh
-rwxr-xr-x 1 matheus devs 512 Mar 26 script.sh
# ou numericamente: 755 = rwx r-x r-x
$ chmod 600 ~/.ssh/id_ed25519
-rw------- 1 matheus matheus 411 Mar 26 id_ed25519
# 600 = só o dono lê/escreve (obrigatório para chaves SSH)
chown & chgrp
chown user fileMuda o dono do arquivo
chown user:group fileMuda dono e grupo simultaneamente
chown -R user:group dir/Aplica recursivamente
chgrp group fileMuda apenas o grupo
idExibe UID, GID e grupos do usuário atual
groups userLista grupos de um usuário
leitura de permissões
$ ls -la arquivo.txt
-rw-r--r-- 1 matheus devs 1024 Mar 26 10:00 arquivo.txt
# tipo dono grupo outros
# d=dir | -=arquivo | l=link
ACLs — Controle de Acesso Estendido
getfacl fileExibe ACLs do arquivo
setfacl -m u:alice:rw fileDá leitura/escrita para alice
setfacl -m g:devs:rx dirDá leitura/execução ao grupo devs
setfacl -x u:alice fileRemove ACL de alice
setfacl -b fileRemove todas as ACLs do arquivo
setfacl -R -m u:alice:rX dir/Aplica recursivamente
Requisito: O filesystem deve estar montado com a opção acl (já padrão no ext4/xfs modernos).

📜Shell Scripting
Variáveis, condicionais, loops, funções e tratamento de erros em Bash.
Estruturas Básicas
script.sh
#!/usr/bin/env bash
# Variáveis
NAME="wisecort"
echo "Olá, ${NAME}!"

# Condicional
if [[ -f "/etc/os-release" ]]; then
  source /etc/os-release
  echo "Distro: ${NAME}"
elif [[ -d "/tmp" ]]; then
  echo "/tmp existe"
else
  echo "Nada encontrado"
fi

# Loop for
for i in {1..5}; do
  echo "Item ${i}"
done

# Loop while
count=0
while [[ ${count} -lt 3 ]]; do
  echo "count=${count}"
  ((count++))
done
Funções & Tratamento de Erros
functions.sh
# Sair imediatamente em caso de erro
set -euo pipefail

# Função com retorno
greet() {
  local user="${1:-world}"
  echo "Hello, ${user}!"
}

greet "wisecort"

# Tratamento de erros com trap
cleanup() {
  echo "Limpando temporários..."
  rm -f /tmp/script_$$.*
}
trap cleanup EXIT ERR

# Arrays
files=("a.txt" "b.txt" "c.txt")
for f in "${files[@]}"; do
  echo "Arquivo: ${f}"
done

Criando um Unit File
/etc/systemd/system/meuapp.service
[Unit]
Description=Meu Aplicativo
After=network.target
Wants=network.target

[Service]
Type=simple
User=www-data
WorkingDirectory=/opt/meuapp
ExecStart=/opt/meuapp/start.sh
ExecReload=/bin/kill -HUP $MAINPID
Restart=on-failure
RestartSec=5s
StandardOutput=journal
StandardError=journal

[Install]
WantedBy=multi-user.target

🔎Regex — Expressões Regulares
Metacaracteres, quantificadores, grupos e exemplos práticos com grep, sed e awk.
Metacaracteres Essenciais
.Qualquer caractere (exceto newline)
^Início da linha
$Final da linha
\dDígito (0-9) — equivale a [0-9]
\wPalavra (letra, dígito, _) — [a-zA-Z0-9_]
\sEspaço em branco (espaço, tab, newline)
\D \W \SNegação de \d \w \s respectivamente
[abc]Um caractere: a, b ou c
[^abc]Qualquer caractere exceto a, b, c
[a-z]Range: letras minúsculas de a a z
\bBoundary — limite de palavra
\Escapa o próximo caractere especial
Quantificadores
*Zero ou mais vezes (greedy)
+Uma ou mais vezes
?Zero ou uma vez (opcional)
{n}Exatamente n vezes
{n,}No mínimo n vezes
{n,m}Entre n e m vezes
*? +? ??Versão lazy (não-greedy) dos quantificadores

Grupos & Alternância
(abc)Grupo de captura
(?:abc)Grupo sem captura
(?P<nome>abc)Grupo nomeado
a|bAlternância — a ou b
\1 \2Backreference — referencia grupo 1, 2…
(?=abc)Lookahead — seguido de abc
(?!abc)Negative lookahead — não seguido de abc

grep examples
# Linhas que começam com "ERROR"
$ grep -E '^ERROR' app.log

awk com Regex
awk examples
# Imprime linhas onde campo 3 > 1000

# Soma coluna 2 e exibe total
$ awk '{soma+=$2} END{print "Total:", soma}' vendas.txt

# Filtra por regex e imprime campo específico
$ awk '/^ERROR/{print $2, $NF}' app.log
2024-03-26 timeout

# Conta ocorrências por valor do campo 1
$ awk '{count[$1]++} END{for(k in count) print count[k], k}' access.log | sort -rn
142 192.168.1.10
87 10.0.0.1

# Muda separador de campo
$ awk -F: '{print $1, $3}' /etc/passwd
